

<h1>🚀 Sorteador de Numeros<h1/>
<br>

📖 Sobre o Projeto:

<h2>Fiz esse projeto com objetivo de Sortear numeros aleatorios entre os valores inseridos, um projeto que aprendi no curso do DevClub- Front-end. Apenas peguei a base na aula e fui pesquisando e codando até chegar nesse resultado, foi um projeto que gostei muito.
</h2>
<br>
<h2>🛠 Tecnologias Utilizadas<h2/>
  <br>

<h2>✨ Como interagir:</h2>
<br>
<ul>
  <li>1 - Digite nas caixas o valor minimo e maximo que você deseja que seja sorteado.</li>
  <li>2 - Verifique se o valor max não é menor que o minimo, pois se for você recebera uma mensagem de erro</li>
  <li>3 - Ja com os valores inseridos apenas clique no botão "<strong>Sortear<strong/>"</li>
  <li>E pronto, seu numero ira aparecer na tela</li>
</ul>
